# Практика №23. Контейнеризация приложений с Docker

## Описание

Реализована система балансировки нагрузки для веб-приложения с полной контейнеризацией через Docker Compose:

- 3 backend-сервера на Node.js (Express) в отдельных контейнерах
- Балансировщик Nginx в отдельном контейнере
- Все сервисы объединены в одну сеть Docker
- Настроена отказоустойчивость через `max_fails` и `fail_timeout`

## Запуск

### 1. Клонировать репозиторий или перейти в папку проекта

```bash
cd practice_23
```

### 2. Запустить 

```bash
docker compose up --build -d
```

### 3. Проверить, что контейнеры запущены

```bash
docker ps
```
